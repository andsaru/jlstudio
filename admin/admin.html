Acceso administrador

Entra con tu usuario de Supabase para gestionar la galería.

Subir nuevas imágenes

Selecciona una o varias imágenes. El sistema generará una miniatura automática para la galería y conservará una versión grande para el lightbox.

Imágenes de la galería

Arrastra desde la miniatura para reordenar. También puedes editar título, alt, visibilidad o borrar imágenes.
